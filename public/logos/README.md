# Company Logo

Place your company logo image file in this directory with one of these filenames:

- `company-logo.png` - Preferred format (supports transparency)
- `company-logo.jpg` - Alternative format

## Logo Requirements:
- **Format:** PNG (preferred) or JPG
- **Size:** Recommended 200x200px to 300x300px
- **Shape:** Square or circular logo works best
- **Background:** Transparent (PNG) or white background
- **File size:** Keep under 200KB for optimal PDF performance

## Notes:
- The logo will be displayed in a circular container (45px x 45px)
- If logo image is not found, "GC" text will be displayed as fallback
- Logo will appear on both pages of the quotation PDF
