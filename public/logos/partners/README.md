## Image Requirements:
- Format: PNG or JPG
- Recommended size: 150-200px width, maintain aspect ratio
- Background: Transparent (PNG) or white background
- File size: Keep under 100KB per image for optimal PDF performance

## Notes:
- If logo images are not found, the PDF will display text labels instead
- Images will be automatically resized to fit within the PDF layout
- Maximum height: 25px in the PDF
